# 🏝️ Hệ Thống Quản Lý Tour Du Lịch

## 📌 Giới thiệu
Đây là hệ thống **quản lý tour du lịch** được xây dựng nhằm hỗ trợ doanh nghiệp quản lý dữ liệu tour, khách hàng, hóa đơn và thống kê báo cáo một cách hiệu quả.

Hệ thống được phát triển phục vụ mục đích học tập và thực hành lập trình web với Python Flask.

---

## ✨ Tính năng chính

- 🔐 Đăng nhập hệ thống (Login)
- 👤 Phân quyền người dùng
- 📦 Quản lý dữ liệu (CRUD: thêm, sửa, xóa, xem)
- 🧾 Xử lý nghiệp vụ hóa đơn
- 📊 Thống kê và báo cáo
- 🔍 Tìm kiếm và lọc dữ liệu

---

## 🛠️ Công nghệ sử dụng

- **Backend:** Python (Flask)
- **Database:** SQL Server
- **Frontend:** HTML
- **IDE:** PyCharm
